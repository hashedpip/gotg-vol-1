# 🚀 Quick Start

📄 Clone or fork `gotg-vol-1`:

```sh
git clone https://github.com/hashedpip/gotg-vol-1.git
```

💿 Install all dependencies:

```sh
cd gotg-vol-1
yarn install
```

🚴‍♂️ Run your App:

```sh
yarn run serve
```

A static server will start and be served at http://localhost:9011

🚚 Building the project for distribution:

```sh
yarn run build
```
